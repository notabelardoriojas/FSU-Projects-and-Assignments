Instructions for running syn.m

Images to be read have to be 2^n by 2^n pixels where n is any integer. 
Sounds are saved as a file called "sample.wav"
You can use the command "sound(bigwave,fs)" to play the sound in MatLab.

Test images are supplied.
	red, blue, and green, will produce the wave corresponding to each waveform.
	gradient16 will produce the sound associated with a red to blue gradient.
	testX is the sounds of various patterns/random colors.
	choiceA/B/C are related to the presentation.
